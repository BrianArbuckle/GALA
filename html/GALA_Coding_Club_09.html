Geographic Coordinate System¶

A geographic coordinate system is a coordinate system used in geography that enables every location on Earth to be specified by a set of numbers, letters or symbols. The coordinates are often chosen such that one of the numbers represents a vertical position, and two or three of the numbers represent a horizontal position. A common choice of coordinates is latitude, longitude and elevation.

For a 2d map, all we need is latitude, longitude.

Latitude¶

Is the angular distance of the places in North and South direction of Earth equator.

All parallel circles from the equator up to the poles are called parallels of latitudes. Latitudes are measured in degrees. Equator is the 0 of latitude. Everything north to the north pole is a positive number up to 90, while everything south is a negative number.

Longitude¶

Is the angular distance of the places in East and West direction of the Prime Meridian, which is in Greenwich London, England. Prime Meridian, the 0 of longitude.

Folium & Leaflet¶

Python libraies or packages.¶

Like matplotlib, there are any graphics packages. Folium is a realy cool one!

Folium builds on the data wrangling strengths of the Python ecosystem and the mapping strengths of the Leaflet.js library. Manipulate your data in Python, then visualize it in on a Leaflet map via Folium. Leaflet is a widely used open source JavaScript library used to build web mapping applications.

The nice thing for us, it does some much work behind the scenes. And with out GALA latitude and longitude, we can plot a map of our location.
